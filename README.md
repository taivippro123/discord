## 💬 Discord Clone

A real-time chat application inspired by Discord.

### 🛠️ Tech Stack

- ⚛️ **Frontend**: ReactJS  
- 🟩 **Backend**: Node.js  
- 🐬 **Database**: MySQL

### ☁️ Deployment & Storage

- 🚀 **Frontend Hosting**: Vercel  
- 🔧 **Backend Hosting**: Render  
- 🐘 **Database Hosting**: Cleaver Cloud  
- 📦 **Media Storage**: Cloudinary (images & videos)
